# Apple Vision Pro Website

A modern, interactive website showcasing Apple Vision Pro with smooth scrolling animations and immersive visuals.

## Features

- Smooth locomotive scroll animations
- GSAP-powered scroll triggers
- Interactive canvas animations
- Responsive design
- Video backgrounds
- 360-degree product views

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Locomotive Scroll
- GSAP (GreenSock Animation Platform)
- ScrollTrigger

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required - it's a static website

## Deployment

This project is deployed on Vercel for fast, global CDN delivery.
